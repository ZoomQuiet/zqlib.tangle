要求:
- < 30行
- 最好可以运行(不要求正确)
- 包含最多的含义
- 任何方面的觉悟!
- 最长的适用期(考虑到语言的变迁,最好甭用将来一定没落的语言)
- 包含:
  + 设计者联系
  + 出处,或是发源地
  + 代码文本
  + 颜色建议
  + 运行截屏
